Relatório de Estágio

Introdução

Este relatório tem como objetivo apresentar as atividades desenvolvidas durante o estagio.

Objetivos

Adquirir experiência prática na área de desenvolvimento de web.

Objetivo Geral

Desenvolver habilidades e conhecimentos na criação de sites e aplicações web.

Objetivos Específicos

Aprender a linguagem HTML, CSS e JavaScript

Atividades realizadas

No período de estágio, foram realizadas diversas atividades relacionadas ao desenvolvimento de web

Atividade 01

Criação de um site institucional para um cliente utilizando HTML, CSS e JavaScript

Atividade 02

Desenvolvimento de um projeto contendo as aplicações de desenvolvimento de web

Conclusão

O estágio me concedeu um aprimoramento nos meus conhecimentos em desenvolvimento de web
